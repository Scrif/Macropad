# Macropad
 A custom macropad with 3D printed case and custom PCB.

Process:
1. Define requirements.
  - 3x3 key matrix
  - 3 switched rotary encoders at the top
  - LED lighting (of course)
  - Hotswappable switches
  - USB-C connector
  - RP2040 MCU
2. Design PCB
3. Design case
4. Order PCBA
5. Print case.
6. Assemble and test!
